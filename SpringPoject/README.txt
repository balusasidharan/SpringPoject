This is sample project is to help people who are looking for sample Spring
project which has Hibernate  and Active MQ Integration

Prerequisites
1. This project runs without web.xml. Servlet Specification 3.0 is used . So you
need a servlet3.0 compatible web container
2.Uses postgreSQL database 
3.The database that this application uses is "springdb" . So you will have to create a database in that name. If you want to edit the user name and password you can edit it
in the source file
4.You will need Active MQ installed on your machine
